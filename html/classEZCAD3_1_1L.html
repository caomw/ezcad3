Detailed Description

L corresponds to a length.

This class solves the issue mixed units. Internally, all lengths are turned into millimeters. 

A *L* represents a length.

Parameters
selfis the L object to initialize.
mmis the number of millimeters to add in.
cmis the number of centiimeters to add in.
inchis the number of inches to add together in.
ftis the number of feet to add together in.

__init__() will initialize self (an *L object) to contain the sum of mm, cm, inch, and ft. Usually, only one of these values is specfied. The argment values must be either an int or float. The inch parameter can also be specified as a string of the form "W-N/D" where is a whole number, N is a numerator, and D is a denumerator. In this case, the result is computed as float(W)+float(N)/float(D).
